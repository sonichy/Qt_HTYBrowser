快捷键

F1关于和帮助
F5刷新当前页
Esc停止加载当前页
Ctrl + T新建标签页
Ctrl + W关闭当前标签页
Ctrl + O打开本地文件
Ctrl + 回车补全网址：http://www. + 地址栏字符串 + .com
Alt + ←跳转上一个网页
Alt + →跳转下一个网页
Ctrl + Tab向右切换标签

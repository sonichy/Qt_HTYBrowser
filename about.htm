快捷键

F1关于和帮助
F5刷新当前页
F11全屏、退出全屏
Esc停止加载当前页
F12开发者工具
Ctrl + T新建标签页
Ctrl + W关闭当前标签页
Ctrl + O打开本地文件
Ctrl + 回车补全网址：http://www. + 地址栏字符串 + .com
Alt + ←跳转上一个网页
Alt + →跳转下一个网页
Ctrl + Tab向右切换标签
Ctrl + +放大当前页
Ctrl + -缩小当前页
Ctrl + 0当前页恢复原始大小
